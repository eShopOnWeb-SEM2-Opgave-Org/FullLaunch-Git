
# Hvordan skal jeg bruge dette projekt?

Før man kør noget som helst andet, skal man først køre setup.bat.

Dette er fordi dette projekt gør brug af submodules, og de skal sættes op inden det kan bruges.

Ud over dette gør CatalogMicroservice brug af en dårlig måde at tilgå sine InventoryMicroservice NuGets.
Dette kommer at at der var nogel problemer med vores devops NuGet feed.
